## Setup

After clone or download this project.

Install Ionic CLI

```
$ npm install -g ionic
```

Install dependencies

```
$ cd <project>
$ npm install
```

Run the app on Localhost

```
$ ionic serve
```

See more at [Get started with Ionic](https://ionicframework.com/docs/v3/)
